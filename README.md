## Installation

Para ejecutar el backend ejecuta:

```bash
$./backend/mvnw spring-boot:run
```

Antes instalar Node Module

```bash
$ ./frontend/npm install
```
Para ejecutar el frontend ejecuta:

```bash
$ ./frontend/ng serve
```
Navegar a http://localhost:4200/
